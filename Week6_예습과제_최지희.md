## 1. Introduction

### 논문이 다루는 분야

- Textual entailment, 질의응답, 의미적 유사도 평가, 문서 분류 등 광범위한 자연어 이해 태스크를 위한 반지도 학습 방법론
- 대규모의 레이블이 없는 텍스트 말뭉치로 언어 모델을 Generative pre-training시킨 후, 각 타겟 태스크에 맞춰 지도 학습 기반으로 미세 조정하는 범용적인 프레임워크 연구

![image.png](attachment:cb0cd760-f4de-47df-8810-ba3d475483af:image.png)

### 해당 task에서 기존 연구 한계점

- **레이블 데이터의 심각한 부족**
 대부분의 딥러닝 방법은 수동으로 레이블링된 방대한 데이터가 필요하지만, 특정 목표 태스크를 위한 레이블 데이터는 매우 부족하여 모델이 적절한 성능을 내기 어려움
- **최적의 학습 목적 함수 부재**
 레이블이 없는 텍스트에서 단어 수준을 넘어선 Higher-level semantics를 학습할 때, 어떤 목적 함수가 전이 학습에 가장 효과적인지 명확한 합의가 없었음
- **복잡한 모델 구조 변경의 번거로움**
 학습된 표현을 타겟 태스크로 전이할 때, 기존 연구들은 태스크마다 모델 아키텍처를 대대적으로 수정하거나 복잡한 학습 체계를 추가해야 하는 비효율성이 존재함

### 논문의 contributions

![image.png](attachment:bf3fbf8a-444e-4d3d-9ef2-9ddb6c9e7f25:image.png)

- 트랜스포머 구조를 활용하여 레이블이 없는 텍스트에서 언어 모델을 사전 학습한 뒤, 최소한의 아키텍처 변경만으로 다양한 타겟 태스크에 미세 조정하는 효율적인 2단계 학습 프레임워크를 성공적으로 구축함
- 미세 조정 단계에서 모델 구조 변경을 피하기 위해, 다중 문장이나 질의응답 같은 복잡한 입력을 단일 연속 시퀀스로 변환하는 입력 변환 기법을 도입함
- 12개의 자연어 이해 벤치마크 중 9개(RACE, Story Cloze Test, MultiNLI 등)에서 태스크별 특화 모델들을 제치고 압도적인 SOTA 성능을 달성함
- 사전 학습된 모델이 미세 조정 없이도 유용한 언어적 지식을 획득하여 다양한 태스크를 수행할 수 있음을 증명하는 제로샷 행동 분석 결과를 제시함

## 2. Related Work

**1) 서론에서 언급한 기존 연구들에 대해 어떻게 서술하는가**

- 초기 연구들은 레이블이 없는 데이터에서 단어 수준의 통계를 계산해 지도 학습의 특징으로 사용함
- 이후 단어 임베딩이 널리 쓰이며 큰 성능 향상을 가져왔으나, 이는 주로 단어 단위의 정보만 전달하는 한계가 있었음을 지적함
- 언어 모델을 활용한 최근 연구들은 사전 학습된 모델의 은닉 상태를 보조 특징으로 활용하는 방식을 취했으나, 이는 타겟 태스크마다 대규모의 새로운 파라미터와 특화된 아키텍처를 추가해야 하는 번거로움이 있다고 서술함
- 본 논문과 가장 유사하게 언어 모델 사전 학습 후 미세 조정을 거치는 선행 연구들이 있었으나, 이들은 LSTM을 사용했기 때문에 짧은 범위의 예측 능력에만 머무른다는 구조적 한계를 짚어냄

**2) 제안 방법의 차별성을 어떻게 표현하고 있는가**

- 단순한 단어 수준을 넘어서 텍스트의 고차원적 의미 체계를 포착하는 것을 핵심 목표로 삼았음을 명시함
- 기존 연구들의 LSTM 대신 트랜스포머 네트워크를 채택하여, 훨씬 더 긴 범위의 언어 구조와 문맥을 성공적으로 포착할 수 있음을 강력한 차별점으로 내세움
- 전이 학습 과정에서 타겟 태스크에 맞춰 아키텍처를 대대적으로 수정할 필요 없이, 아주 최소한의 변경만으로 효과적인 학습이 가능함을 강조함
- 기존 방식들이 주로 단순 텍스트 분류에 국한되어 평가된 것과 달리, 자연어 추론이나 스토리 완성 등 훨씬 광범위하고 복잡한 태스크에서 모델의 범용성을 입증했다는 점을 차별화 포인트로 제시함

## 3. 제안 방법론

### Main Idea

![image.png](attachment:9661e78b-d194-4c78-9fc0-10f93bf9ea59:image.png)

![image.png](attachment:81434f24-4207-4646-af3c-61840876dc42:image.png)

- 대규모 무레이블 말뭉치로 언어 모델을 훈련하는 비지도 사전 학습 단계와, 해당 파라미터를 타겟 태스크의 레이블 데이터에 맞게 업데이트하는 지도 미세 조정 단계로 구성된 2단계 프레임워크를 제안함
- 장기 의존성(Long-term dependencies)을 효과적으로 처리하기 위해 기존의 순환 신경망 대신 트랜스포머 디코더 아키텍처를 채택하였으며, 입력 문맥에 대해 다중 헤드 셀프 어텐션(Multi-headed self-attention) 연산을 수행함
- 미세 조정 단계에서 타겟 태스크의 분류 목적 함수뿐만 아니라 기존의 언어 모델링 목적 함수를 보조로 함께 사용하여, 모델의 범용화 성능을 높이고 수렴 속도를 가속함
- 복잡하고 다양한 형태의 타겟 태스크 입력을 처리하기 위해, 시작 토큰, 끝 토큰, 구분자 토큰을 적절히 끼워 넣어 하나의 연속된 1차원 시퀀스로 평탄화하는 입력 변환 기법을 사용함

### Contribution

![image.png](attachment:048392ea-cb31-43f0-a31c-ab77c512d788:image.png)

- 기존 방법론들이 타겟 태스크마다 모델의 뼈대를 대대적으로 수정해야 했던 구조적 한계를, 단순한 시퀀스 이어 붙이기(Traversal-style approach)로 어떻게 우회하고 극복했는지 논리적 근거와 구체적인 결합 방식을 통해 명확하게 입증함
- 트랜스포머 블록을 통과한 마지막 활성화 값이 선형 출력 레이어와 어떻게 수식적으로 연결되는지 명료하게 서술하였으며, 텍스트 분류, 함의, 의미 유사도, 다중 선택 등 각 문제 유형별로 데이터를 어떻게 이어 붙여야 하는지까지 아주 구체적으로 명시하여 별도의 추측 없이 바로 코드로 구현 가능하도록 친절하게 작성됨

## 4. 실험 및 결과

### Dataset

![image.png](attachment:d7aeb7b9-2504-4a19-a58c-27db3815c4a0:image.png)

- **사전 학습용:** 미출판 도서 7천여 권으로 구성되어 긴 문맥의 흐름을 학습하기에 최적화된 BooksCorpus 사용
- **미세 조정용 (총 12개 벤치마크):**
    - 자연어 추론: SNLI, MultiNLI, QNLI, RTE, SciTail
    - 질의응답 및 상식 추론: RACE, Story Cloze Test
    - 의미 유사도: MRPC, QQP, STS Benchmark
    - 텍스트 분류: SST-2, CoLA

### Baseline

- 각 태스크별로 복잡하고 정교하게 설계된 기존 최고 성능 모델들을 비교군으로 설정함
- 자연어 추론의 ESIM+ELMo, 질의응답의 Dynamic Fusion Net, 텍스트 분류의 Sparse byte mLSTM 등이 대표적인 비교 대상임
- 내부 요인 분석을 위해 트랜스포머 대신 LSTM을 쓴 모델, 사전 학습을 아예 빼버린 모델 등을 자체 베이스라인으로 추가 구축함

### 결과

![image.png](attachment:f22cf0a7-6851-41b2-b55f-5d7dfbd2f8cf:image.png)

- 전체 12개 데이터셋 중 9개에서 앙상블 기법까지 동원된 기존 특화 모델들을 제치고 단일 모델로 SOTA 성능을 달성함
- 특히 긴 문맥 이해가 필수적인 상식 추론에서 8.9%, 질의응답에서 5.7%라는 파격적인 절대 성능 향상을 기록함
- 언어적 직관력을 평가하는 CoLA 태스크에서도 기존 35.0점에서 45.4점으로 뛰어올라 모델이 훌륭한 언어적 편향을 학습했음을 입증함
- 구조 비교 실험 결과 트랜스포머가 LSTM보다 전이 학습에 훨씬 적합함을 입증했으며, 사전 학습을 생략할 경우 전체 성능이 14.8%나 폭락함을 확인하여 사전 학습의 필수성을 증명함
- 제로샷 분석을 통해, 모델이 미세 조정을 거치지 않고도 사전 학습이 진행됨에 따라 스스로 감성 분석이나 질의응답 같은 언어 태스크 수행 능력을 꾸준히 터득하고 있음을 확인함

## 5. 결론 (배운점)

**의의**

- 레이블이 없는 대규모 텍스트로 트랜스포머를 사전 학습한 후, 구조 변경을 최소화하여 다양한 자연어 처리 태스크에 적용하는 범용적인 2단계 프레임워크를 성공적으로 확립함
- 12개 중 9개 벤치마크에서 최고 성능을 기록하며, 자연어 처리의 패러다임을 특정 태스크 맞춤형 모델 설계에서 '거대 범용 모델의 전이 학습'으로 전환하는 결정적 계기를 마련함

**한계**

- 하지만 타겟 태스크로 미세 조정할 때, 입력 데이터 사이에 시작, 끝, 구분자 등 특수한 기호 토큰을 태스크별 규격에 맞게 수동으로 끼워 넣어야 하는 전처리 과정의 번거로움이 여전히 한계로 남음

**좋았던 점**

- 모델의 신경망 구조 자체를 복잡하게 개조하는 대신, 질문과 답변 등을 단순히 하나의 긴 텍스트 시퀀스로 이어 붙이는 직관적이고 훌륭한 발상의 전환으로 한계를 극복한 점이 가장 인상 깊음

**아쉬운 점**

- 방대한 데이터셋과 막대한 컴퓨팅 연산량이 절대적으로 뒷받침되어야 하는 연구라, 학부생 개인 환경에서 베이스라인부터 직접 학습시켜 보며 튜닝해 보기에는 현실적인 스케일의 벽이 크게 느껴짐

Densely Connected Convolutional Networks

## Introduction

### 논문이 다루는 분야

- 시각적 객체 인식을 위한 CNN 아키텍처 설계 분야.
- 특히 네트워크가 깊어짐에 따라 발생하는 정보 및 기울기 흐름 문제를 해결하기 위한 새로운 연결 패턴 연구.

### 해당 task에서 기존 연구 한계점

- **기울기 소실 문제**
네트워크가 점점 깊어지면서 입력 데이터나 기울기 정보가 네트워크의 끝에 도달하기 전에 사라지거나 변질되는 현상이 발생함.
- **정보 흐름의 방해**
 ResNet과 같은 기존 모델들은 층 간 짧은 경로를 만들어 이를 해결하려 했으나, 이전 특징들을 단순히 더하기 연산으로 결합하기 때문에 네트워크 내 온전한 정보 흐름을 방해할 우려가 있음.
- **매개변수의 비효율성**
각 층이 고유의 가중치를 가지고 이전 상태를 보존하려 하다 보니, 중복되는 특징 맵을 불필요하게 다시 학습하게 되어 결과적으로 전체 매개변수 크기가 비효율적으로 커짐.

### 논문의 contributions

![image.png](attachment:630f8ef6-ab63-419c-8909-0fe1394ac68b:image.png)

- 피드포워드 방식으로 모든 층이 그 이후의 모든 층과 직접 연결되는 DenseNet 아키텍처를 제안함.
- 기존의 더하기 방식 대신 특징들을 이어 붙이는 방식을 채택하여, 기울기 소실 문제를 완화하고 특징의 전파 및 재사용을 극대화함.
- 층마다 불필요한 특징 맵을 다시 학습할 필요가 없도록 설계하여, 기존 모델들보다 훨씬 적은 수의 매개변수만으로도 강력한 성능을 내는 높은 효율성을 확보함.
- CIFAR-10, CIFAR-100, SVHN, ImageNet 등 주요 벤치마크 태스크에서 더 적은 연산량과 매개변수만으로 기존의 최고 성능을 뛰어넘는 결과를 입증함.
